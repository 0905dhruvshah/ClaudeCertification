# CCA-F Mock Test Exam

A self-contained static website, ready for Vercel deployment. It includes 40 MCQs, a 120-minute exam timer, question navigation, automatic submission when time expires, and a complete answer review with source justifications.

## Deploy to Vercel

1. Create a GitHub repository and upload this folder's contents.
2. Import the repository at [vercel.com/new](https://vercel.com/new).
3. Leave the framework preset as **Other** and deploy. No build command is required.

You can also deploy using the Vercel CLI from this folder: `vercel --prod`.
